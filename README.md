# 财务数据可视化分析工具

## 项目简介
这是一个基于 Python Streamlit 和 Plotly 的财务数据可视化分析工具。它可以读取本地 Excel 格式的财务报表数据，自动计算滚动（TTM）数据，并生成交互式的增长趋势图表。

## 功能特性
- **多公司支持**：自动扫描数据目录下的公司文件夹。
- **数据自动处理**：
  - 自动读取资产负债表、现金流量表、主要财务指标。
  - 自动将累计季度数据转换为 TTM（滚动12个月）数据。
  - 单位自动转换为“亿元”。
- **交互式图表**：
  - 支持缩放、平移。
  - 悬停查看详细数值。
  - 自定义时间范围筛选。
- **视觉设计**：采用深色主题，配色符合专业财经分析风格。

## 环境要求
- Python 3.8+
- 依赖包见 `requirements.txt`

## 安装说明

1. 克隆或下载本项目。
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

## 使用说明

1. **准备数据**：
   请确保数据文件存放在 `Stock_chart/数据导出/` 目录下。
   目录结构示例：
   ```
   Stock_chart/
   ├── src/
   ├── 数据导出/
   │   └── 比亚迪/
   │       ├── 002594_cash_report.xls
   │       ├── 002594_debt_report.xls
   │       ├── 002594_main_report.xls
   │       └── ...
   ```

2. **启动应用**：
   在项目根目录下运行：
   ```bash
   streamlit run src/app.py
   ```

3. **操作指南**：
   - 左侧侧边栏选择要分析的“公司”。
   - 选择“图表类型”（目前支持“总营收与主业现金流增长趋势”）。
   - 使用“时间范围”滑块调整数据展示区间（默认最近10年）。
   - 右侧主区域展示交互式图表，下方可展开查看详细数据表格。

## 配置说明
可以在 `src/config.py` 中修改：
- **COLORS**: 图表颜色主题。
- **CHARTS**: 新增或修改图表定义。
- **FIELDS**: 修改 Excel 文件列名映射。

## 扩展开发
- **添加新图表**：在 `src/config.py` 的 `CHARTS` 字典中添加新的图表配置即可。
- **数据处理逻辑**：核心逻辑位于 `src/data_loader.py`。
